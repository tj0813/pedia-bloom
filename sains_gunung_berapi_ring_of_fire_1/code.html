Gunung Berapi & Cincin Api

Ayo jelajahi kekuatan dahsyat dari dalam Bumi dan temukan rahasia gunung-gunung api di Indonesia!

Apa itu Gunung Berapi?

Bayangkan Bumi seperti panci presto raksasa. Di bawah kerak tempat kita berdiri, ada cairan batu super panas bernama Magma. Ketika tekanan di dalam panci ini terlalu besar, magma mencari jalan keluar dan... BOOM! Terjadilah gunung berapi.
