Harmoni dalam Gerak

Tari Saman adalah salah satu media untuk menyampaikan pesan atau dakwah. Tarian ini mencerminkan pendidikan, keagamaan, sopan santun, kepahlawanan, kekompakan, dan kebersamaan. Menariknya, tarian ini tidak menggunakan instrumen musik, melainkan menggunakan suara dari para penari dan tepukan tangan mereka.
